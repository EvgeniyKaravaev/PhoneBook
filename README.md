# crudToDo
CRUD
Это приложение телефонный справочник (PhoneBook), реализовано с использованием такого стека технологий:
1. Java
2. Spring Framework ( Data JPA, Boot 2, Security)
3. Hibernate
4. Flyway
5. Lombok
6. PostgreSQL
7. JUnit, Mockito
8. Bootstrap
9. HTML 5
10. CSS3
11. Thymeleaf
